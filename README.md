## Funcionalidades
- Mostrar a lista de produtos a partir de um arquivo JSON
- Pesquisar um produto por nome
- Ordenar por nome, popularidade (score), ou preços
- Adicionar ou remover produtos do carrinho

## Scripts disponíveis

### `yarn start`

### `yarn test`

### `yarn build`

### `yarn eject`
